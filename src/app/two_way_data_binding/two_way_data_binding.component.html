2-WAY DATA BINDING

We add the {{ ngModelString }} directive to the element where we want to use 2-way data binding. Then we set the directive equal to the variable name, which is defined in the typescrpt file for the component we are working in. 

<input type="text" [(ngModel)]="name">

Before this will work, we need import FormsModule in the app.module.ts file. 

import {{ formsModuleString }} from '@angular/forms';

Then we must add the FormsModule to our imports array in the app.module.ts file.

@NgModule({{ openBracketString }}
    declarations: [
        AppComponent,
        TwoWayDataBindingComponent
    ],
    imports: [
        BrowserModule,
        FormsModule
    ],
    providers: [],
    bootstrap: [AppComponent]
{{ closeBracketString }})

{{ name }}
