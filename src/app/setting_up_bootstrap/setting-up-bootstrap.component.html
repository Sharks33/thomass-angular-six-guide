ADDING BOOTSTRAP TO ANGULAR PROJECT

Make sure that your local server is NOT running.
From the project directory run: 

npm install --save bootstrap@3

Inside of the angular.json file, which can be found in the root directory of your project; add: node_modules/bootstrap/dist/css/bootstrap.min.css to the styles array. 

"styles": [
    "node_modules/bootstrap/dist/css/bootstrap.min.css",
    "src/styles.css"
],

Restart your server. 

ng serve
